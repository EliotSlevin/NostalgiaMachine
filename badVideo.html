1. Does the video show an error message like this.

If it does, sorry we can't fix that. Almost every video is blocked in one country or another, if we changed it to work for you, it'd break for somebody else.

2. Is the video private, removed, or has the account been terminated?

Then we can probably fix that.

3. Is the video not the actual song, or the incorrect song?

Then we can also fix that

Fixing through github

Clone the repo, go to app/js/years, and find the year of the broken video. Put in a new youtube video ID and send us a pull request :)

Github is the fastest way for us to fix errors

Fixing via email

Send us an email at swag@swag.com. Please state the year, artist, and song name for each bad video, and supply a youtube link which is working for you :)

Thank you!
